# Materi Instalasi Motor Listrik — Mekanis dan Elektromekanis
## Kelas XI SMK — Fase F

Dokumen ini dipecah menjadi beberapa file agar mudah dibaca oleh sistem/mesin. Sumber asli: `MATERI_INSTALASI_MOTOR_LISTRIK_MEKANIS_DAN_ELEKTROMEKANIS_IML_SMK_4.docx`.

Catatan konversi:
- Seluruh gambar pada dokumen asli digantikan dengan penanda teks `*[Gambar N. Judul Gambar]*` pada posisi aslinya, karena file .txt/.md tidak dapat menyimpan gambar biner. Nomor dan judul gambar dipertahankan sesuai keterangan aslinya.
- Format teks tebal (**bold**) dan miring (*italic*) dari dokumen asli dipertahankan dalam sintaks Markdown.

## Daftar File

1. `01_komponen_mekanis.md` — Bagian MEKANIS: 10 komponen kelistrikan (Saklar TPST, Saklar TPDT, MCB, Kontaktor Magnetik, TOR, TDR, Push Button, Pilot Lamp, NO, NC)
2. `02_jenis_diagram_dasar.md` — Bagian ELEKTROMEKANIS, Sub-bagian A: Jenis-jenis diagram/gambar instalasi (Diagram Blok, Diagram Satu Garis, Diagram Pengawatan, Diagram Lintasan/Aliran)
3. `03_diagram_kendali_elektromagnetik.md` — Bagian ELEKTROMEKANIS, Sub-bagian B: Jenis-jenis diagram gambar instalasi motor listrik dengan kendali elektromagnetik (8 jenis rangkaian kendali, termasuk DOL, dua tempat, berurutan manual/otomatis, forward-reverse, dan star-delta)

## Ringkasan Isi per File

**01 — Komponen Mekanis** membahas komponen-komponen dasar instalasi motor listrik: fungsi, cara kerja, dan penerapannya dalam rangkaian kontrol motor.

**02 — Jenis Diagram Dasar** membahas empat jenis diagram/gambar instalasi listrik secara umum: diagram blok, diagram satu garis, diagram pengawatan, dan diagram lintasan/aliran.

**03 — Diagram Kendali Elektromagnetik** membahas delapan diagram kerja kendali elektromagnetik motor listrik secara rinci, lengkap dengan penjelasan prinsip kerja rangkaian berdasarkan aliran arus untuk tiap jenis rangkaian.
